## Run Locally

Clone the project

```bash
  git clone https://github.com/carlosexojogavilan/barrel-cloud.git
```

Go to the project directory

```bash
  cd barrel-cloud
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
